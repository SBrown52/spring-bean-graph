# Spring Bean Viewer

This projects connects to the Spring Boot actuator bean endpoint (/actuator/beans) and tries to show a dependency graph of all the beans.

100% a work in progress

Written by a back end java dev so excuse some of the UI code :)


This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
